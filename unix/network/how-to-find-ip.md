# How to find ip address in Unix

### With dig
```bash
WEBSITE=www.google.com
dig $WEBSITE
```

## With nslookup
```bash
nslookup $WEBSITE
```

## With host
```bash
host $WEBSITE
```

## With ping
```bash
ping $WEBSITE
```
